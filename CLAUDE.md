# CLAUDE.md — hs-zing

## Overview

Zing is a Hammerspoon Spoon that manages keyboard shortcuts and provides
quick web search functionality.

## Commit Conventions

Use [Conventional Commits](https://www.conventionalcommits.org/) format:

```
<type>(<scope>): <description>
```

Types: `feat`, `fix`, `chore`, `docs`, `refactor`, `test`, `style`, `perf`

Scope is optional but encouraged (e.g. `fix(search): ...`, `feat(bindings): ...`).

Include the issue number when applicable (e.g. `feat: add search provider (#2)`).

## Branch Naming

Use the same type prefixes as commits, followed by a short description:

```
<type>/<short-description>
```

Examples: `feat/search-provider`, `fix/keybinding-conflict`, `chore/update-deps`
